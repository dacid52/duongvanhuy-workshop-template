**Tuần 1:** [Tìm hiểu tổng quan nền tảng AWS, làm quen với môi trường tài khoản, xây dựng lộ trình học tập, nắm các nguyên tắc kiểm soát chi phí, và hoàn thành các bài lab nhập môn.](1.1-week1/)

**Tuần 2:** [Nghiên cứu kiến trúc Amazon VPC và các thành phần mạng liên quan, cấu hình tài nguyên mạng lõi, thiết lập Security Group và NACL, khởi tạo EC2, đồng thời tìm hiểu các phương án kết nối như VPN, VPC Peering và Transit Gateway.](1.2-week2/)

**Tuần 3:** [Triển khai EC2 trên các subnet, kiểm tra kết nối nội bộ, cấu hình NAT Gateway và EC2 Instance Connect Endpoint, thiết lập mô hình Hybrid DNS với Route 53, và thực hành AWS Backup cho EC2 và EBS.](1.3-week3/)

**Tuần 4:** [Tìm hiểu AWS Storage Gateway kết nối với Amazon S3 để đồng bộ dữ liệu, kiểm tra cơ chế cache và phân quyền truy cập, đồng thời triển khai static website trên Amazon S3 với các cấu hình public access cần thiết.](1.4-week4/)

**Tuần 5:** [Tìm hiểu chuyên sâu về Amazon EC2, thực hành với Windows Server 2025 và Amazon Linux 2023, xem xét tiêu chí lựa chọn instance, tăng cường cấu hình VPC và Security Group, đồng thời làm quen với EBS Snapshot, Systems Manager và IAM.](1.5-week5/)

**Tuần 6:** [Nghiên cứu IAM User, Role, Policy; quản lý dữ liệu trên Amazon S3; làm quen với AWS Cloud9, AWS STS, EC2 Instance Profile, CloudWatch, và theo dõi chi phí cùng AWS Credits.](1.6-week6/)

**Tuần 7:** [Tìm hiểu Amazon RDS, kết nối cơ sở dữ liệu được quản lý với EC2, giám sát tài nguyên bằng CloudWatch Dashboard và Alarms, đồng thời nghiên cứu Auto Scaling, Elastic Load Balancer và đánh giá hiệu năng hệ thống.](1.7-week7/)

**Tuần 8:** [Tiếp tục thực hành với Amazon S3, hoàn thiện chính sách truy cập, triển khai website tĩnh, tìm hiểu Amazon CloudFront để tối ưu tốc độ truy cập, và xem xét các khả năng sao lưu, đồng bộ, phục hồi dữ liệu.](1.8-week8/)

**Tuần 9:** [Nghiên cứu mô hình triển khai có tính sẵn sàng cao trên AWS, tiếp tục giám sát với CloudWatch, tìm hiểu CloudTrail và Systems Manager, đồng thời ôn tập và hệ thống hóa kiến thức đã học.](1.9-week9/)

**Tuần 10:** [Thiết kế kiến trúc triển khai trên AWS, hoàn thiện sơ đồ tổng thể cho các lớp tính toán, cơ sở dữ liệu, lưu trữ và mạng, chuẩn bị môi trường đích, và hoàn tất tài liệu thiết kế.](1.10-week10/)

**Tuần 11:** [Triển khai môi trường ứng dụng trên AWS, cấu hình máy chủ, cơ sở dữ liệu và lưu trữ, kiểm thử tích hợp, xử lý lỗi phát sinh trong quá trình triển khai, và rà soát cấu hình cùng bảo mật.](1.11-week11/)

**Tuần 12:** [Hoàn thiện triển khai, đánh giá kết quả thực hiện, kiểm tra hiệu năng và cấu hình, hoàn chỉnh báo cáo kỹ thuật, và chuẩn bị nội dung trình bày tổng kết.](1.12-week12/)
